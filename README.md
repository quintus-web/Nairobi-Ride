🚌 NairobiRide — Navigate Nairobi Like a Local
NairobiRide is a web-based transit navigation platform designed to bridge the information gap for commuters in Nairobi. Whether you are a newcomer from upcountry, a tourist, or a local exploring a new neighborhood, NairobiRide helps you find matatu route numbers, boarding stages, and estimated fares with ease.

🚀 The Problem
Nairobi's matatu system is vibrant but decentralized. Newcomers often struggle to know:

Which matatu number goes to their destination?

Where exactly is the boarding stage (e.g., is it at Railways, Ambassador, or Koja)?

How much is the typical fare?

What are the stops along the way?

NairobiRide fixes this by providing a centralized, community-updated database of transit information.

✨ Features
🔍 Smart Route Search: Search by destination (e.g., "Ngong") or route number (e.g., "111").

🗺️ Interactive Maps: Visualized routes using Leaflet.js with pins for every stop and a connecting polyline.

💰 Fare Estimates: Real-time range of costs (Min/Max KES).

📍 Stage Highlights: Clearly marked major hubs and minor drop-off points.

🛡️ Admin Dashboard: A powerful backend for managing routes and verifying community contributions.

🛠️ Tech Stack
Backend: Django 6.0 (Python)

Frontend: Bootstrap 5 (Responsive UI)

Mapping: Leaflet.js & OpenStreetMap

Database: SQLite (Development) / PostgreSQL (Production)
